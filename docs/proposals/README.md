# Proposals

Documents in this folder are **proposals**, not committed work. They
sketch directions we might take but haven't decided on. A proposal
graduates by moving into `docs/BUILD_PLAN.md` as a milestone (or by
being marked `rejected` with a one-line reason and left in place for
the record).

Each proposal carries a `Status:` field — `proposed` / `accepted` /
`in-progress` / `done` / `rejected (reason)`. Read that first before
treating anything in here as a plan of record.
