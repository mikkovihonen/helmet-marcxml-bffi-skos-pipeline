# Completed

Plans whose definition of done has been met. Every phase has a
filled-in `Phase commit` hash. These documents stay here as the
audit trail of what shipped and how; the canonical reference for
"what landed when" is the per-phase commit hash inside each plan,
not any external checklist.

Do not edit completed plans except to fix typos or to add a short
post-mortem section if reality diverged meaningfully from what the
plan predicted.
