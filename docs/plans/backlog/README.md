# Backlog

Plans that have been drafted but not yet started. No phase commit
has landed for them. A plan in here is ready to be picked up; it
should be runnable cold by the person executing it (if it isn't,
that's a draft-quality bug — fix the plan, don't change the
folder).

When work on a plan begins, `git mv` it into
[`../in-progress/`](../in-progress/) in the same commit that lands
the first phase commit.
