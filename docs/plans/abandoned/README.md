# Abandoned

Plans that were dropped before completion. Each plan in here
carries an `Abandonment reason` section near the top explaining
why work stopped and what (if anything) supersedes it.

These documents are kept for the historical record — a plan that
was rejected for a documented reason is more useful than one
silently deleted, because the same idea will probably resurface
later and the reasoning saves the conversation.

Do not edit abandoned plans except to fix typos or to add a
forward-pointer to a superseding proposal / plan.
